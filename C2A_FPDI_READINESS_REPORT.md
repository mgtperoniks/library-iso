## 4. Kompatibilitas Versi PDF (PDF Version Compatibility)

* **PDF 1.3:** 0 berkas
* **PDF 1.4:** 0 berkas
* **PDF 1.5:** 25 berkas (100%)
* **PDF 1.6 - 1.7:** 0 berkas

### Analisis Parser FPDI:
FPDI Community Edition (gratis) secara bawaan hanya mendukung PDF versi 1.4 kebawah. Untuk versi 1.5 ke atas, FPDI berpotensi melempar pengecualian:
`This PDF document probably uses a compression technique which is not supported by the free parser.`
Hal ini terjadi karena PDF 1.5 memperkenalkan fitur *Compressed Object Streams* dan *Cross-Reference Streams* yang tidak didukung parser bawaan PHP.
*Solusi Mitigasi:* Apabila pustaka FPDI versi gratis gagal memproses PDF, kita perlu menyiapkan integrasi CLI (seperti **Ghostscript** atau **QPDF**) untuk menurunkan versi kompresi PDF secara otomatis ke v1.4 sebelum diolah oleh FPDI.

1. **Teknis Blocker:** Tidak ada blocker utama. Namun, keterbatasan FPDI gratis terhadap PDF 1.5+ wajib diantisipasi dengan penyiapan fallback menggunakan Ghostscript/QPDF untuk dekompresi format berkas.

3. **Keamanan Margin:** Aman untuk SOP/IK, memerlukan penyesuaian teks 1-baris tipis untuk tipe formulir padat.
4. **Rekomendasi Desain:** Desain B (Standard) dalam format **1-baris horizontal** di bagian margin bawah.
